# TwidyTools Frontend

Modern document and image converter built with Next.js 14, React 18, and Tailwind CSS.

## Features

- ✨ Beautiful, responsive UI
- ⚡ Fast and optimized
- 🎨 Dark mode design
- 📱 Mobile-friendly
- 🔒 Secure file handling

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
cd frontend
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Environment Variables

Create `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:5000
```

## Project Structure

```
frontend/
```
